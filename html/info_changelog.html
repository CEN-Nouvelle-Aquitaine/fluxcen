FluxCEN 4.3

Améliorations:

Améliorations diverses (meilleure gestion des couches PostGIS / intégration de nouveaux messages d'erreurs / correction de bugs / intégration de nouveaux flux)

Détection et gestion automatique des flux CEN-NA sécurisés

Intégration d'une colonne "Métadonnées"

Intégration d'une catégorie intitulée "Couches modifiables" permettant de collaborer (ajout, édition, supression de données) sur une couche PostGIS

Suppression des flux drone (intégration de ces données dans LoaderCEN)

Intégration d'un message d'erreur si tentative d'utilisation de FluxCEN sans connexion internet

Possibilité d'utiliser FluxCEN sans compte CEN-NA pour tous les flux sauf les flux FoncierCEN (flux sécurisés)

Possibilité de modifier les styles des flux directement depuis GitHub

Correction de certains bugs mineurs

Intégration de nouveaux flux WFS & WMS:

  • intégration des données d'occupation du sol de PIGMA 2020 pour l'ensemble de la Nouvelle-Aquitaine
  • création de deux nouvelles catégories "géologie" et "risques" regroupant les flux publics mis à disposition par le BRGM
  • création d'une catégorie "géodésie" regroupant les données de géodésie et de nivellement du Réseau Géodésique Français de l'IGN
